## Shorthand Expansion

These shorthands decompose into longhand properties, allowing individual overrides:

| Shorthand | Expands To |
|-----------|-----------|
| `padding` (p) | padding-top, padding-right, padding-bottom, padding-left |
| `margin` (m) | margin-top, margin-right, margin-bottom, margin-left |
| `gap` | row-gap, column-gap |
| `border-width` (bw) | border-top-width, border-right-width, border-bottom-width, border-left-width |

Multi-value rules (same as CSS):
- 2 values: `p10px-20px` = top/bottom: 10px, right/left: 20px
- 3 values: `p10px-20px-30px` = top: 10px, right/left: 20px, bottom: 30px
- 4 values: `p10px-20px-30px-40px` = top: 10px, right: 20px, bottom: 30px, left: 40px

## Negative Values (neg- prefix)

### Allowed cssProperties
`margin`, `margin-top`, `margin-right`, `margin-bottom`, `margin-left`, `top`, `right`, `bottom`, `left`, `letter-spacing`, `word-spacing`, `text-indent`, `outline-offset`

### Allowed Special Properties
`z-index`, `order`, `letter-spacing`, `transform`

Examples: `neg-mt10px` = margin-top: -10px, `neg-zi5` = z-index: -5, `neg-order1` = order: -1
